# 🌦 Weather App

A simple weather application built using HTML, CSS and JavaScript.

## Features
- Search weather by city
- Real-time API data
- Responsive design
- Clean UI

## Technologies Used
- HTML
- CSS
- -Tailwind css
- JavaScript
- Weather API

## How to Run
1. Download the project
2. Open WeatherApi.html in browser
